⚙️ OneDrive Setup

You only need one value — a Microsoft App ID. Takes 2 minutes to get.

  1. Go to portal.azure.com → App registrations
  2. Click + New registration
  3. Name: anything (e.g. SecureDiary)
  4. Supported account types: Personal Microsoft accounts only
  5. Redirect URI: Single-page application (SPA) →
  6. Click Register → copy the Application (client) ID
A UUID like: 12345678-abcd-...
🔒 Saved only in your browser.
